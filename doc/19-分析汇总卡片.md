## 实现要点

### 核心功能点

1. **时间窗口显示**
   - 显示时间窗口的天数和小时数
   - 格式：`X Days Y Hours` (英文展示)
   - 根据起止日期和时刻计算

2. **平均天气数值统计**
   - 根据产品类型显示不同的统计方式：
     - 日内产品：显示 avg. hourly value
     - 周度或月度产品：显示 avg. daily value
   - 计算逻辑：由 `useWeatherStatistics` 提供气象均值

3. **总天气数值统计**
   - 显示时间窗口内的总数值
   - 计算逻辑：由 `useWeatherStatistics` 提供气象总量

4. **风险事件数量统计**
   - 仅在指定了保险产品后显示
   - 由 `useRiskAnalysis` 提供统计数据

5. **风险严重程度**
   - 仅在指定了保险产品后显示
   - 由 `useRiskAnalysis` 根据风险事件的最高级别计算

### 技术实现方案

有关中心化风险计算及天气数据解耦的详细方案，请参阅：[风险计算架构升级方案](./补充-风险计算架构升级方案.md)

#### 1. 统计数据聚合
统计数据由两个中心化 Hook 协同产出：
- `useWeatherStatistics`: 产出基础气象指标（Time Window, Avg, Total）。
- `useRiskAnalysis`: 产出风险相关指标（Risk Events Count, Severity）。

```typescript
// 1. WeatherStatistics (天气基础统计)
interface WeatherStatistics {
  timeWindow: { days: number; hours: number; totalHours: number; };
  metrics: {
    total: number;
    avgDaily: number;
    avgHourly: number;
  };
}

// 2. RiskStatistics (风险分级统计)
interface RiskStatistics {
  total: number;
  byLevel: { tier1: number; tier2: number; tier3: number; };
  severity: 'low' | 'medium' | 'high' | 'none' | '-';
}
```

#### 2. 分析汇总卡片组件
```typescript
// 伪代码示例
function AnalysisSummary({
  statistics,        // RiskStatistics
  weatherStatistics, // WeatherStatistics
  selectedProduct
}: AnalysisSummaryProps) {
  return (
    <div className="analysis-summary grid grid-cols-2 md:grid-cols-4 gap-4">
      {/* 时间窗口卡片 */}
      <SummaryCard
        title="Time Window"
        value={`${weatherStatistics.timeWindow.days} Days ${weatherStatistics.timeWindow.hours} Hours`}
        icon={<Clock />}
      />
      
      {/* 平均天气数值卡片 */}
      <SummaryCard
        title={selectedProduct?.type === 'daily' ? 'Avg. Hourly Value' : 'Avg. Daily Value'}
        value={`${selectedProduct?.type === 'daily' 
          ? weatherStatistics.metrics.avgHourly 
          : weatherStatistics.metrics.avgDaily} units`}
        icon={<Thermometer />}
      />
      
      {/* 总天气数值卡片 */}
      <SummaryCard
        title="Total Value"
        value={`${weatherStatistics.metrics.total} units`}
        icon={<Activity />}
      />
      
      {/* 风险事件数量卡片（仅在选中产品时显示） */}
      {selectedProduct && (
        <SummaryCard
          title="Risk Events"
          value={statistics.total.toString()}
          icon={<AlertTriangle />}
          severity={statistics.severity}
        />
      )}
    </div>
  );
}
```

## 注意事项

### 常见问题
1. **统计数据计算不准确**

   - 确保数据格式正确
   - 验证计算结果

2. **产品类型判断错误**
   - 确保产品类型正确传递
   - 确保条件判断逻辑正确

3. **时间窗口计算错误**
   - 确保日期和时刻计算正确
   - 处理时区问题（如果需要）

### 技术难点

- **条件渲染**：需要根据产品类型动态调整显示内容
